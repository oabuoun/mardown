For more information: visit [basic syntax](https://www.markdownguide.org/basic-syntax) and [extended syntax](https://www.markdownguide.org/extended-syntax)

```markdown
# Heading - H1
```

**Code:**


**Syntax Wanted: Fenced Code Block**

```

  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}
```

**Code:**

```markdown

  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}
```
